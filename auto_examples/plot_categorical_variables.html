Plotting quantities from a CSV fileÂ¶

This example loads from a CSV file data with mixed numerical and categorical entries, and plots a few quantities, separately for females and males, thanks to the pandas integrating plotting tool (that uses matplotlib behind the scene).

Python source code: plot_categorical_variables.py

import pandas



import matplotlib.pyplot as plt



data = pandas.read_csv('brain_size.csv', sep=';')

gender_data = data.groupby('Gender')

gender_data.boxplot(column=['FSIQ', 'VIQ', 'PIQ'])



from pandas.tools import plotting

plotting.scatter_matrix(data)



plt.show()
